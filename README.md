# eleme-cli
A Command-Line Ele.me Client

## Setup

 - ```mv .env.example .env```
 - ```vi .env```
 - Login to [ele.me](https://ele.me) , Find Cookie Field ```SID``` and input it to ```.env``` file
 - Input Your GPS Location to ```.env``` File

## Example ```.env``` File

```
SID=
longitude=
latitude=
```
